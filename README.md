# Lost and Found App - Geo Features Edition

A comprehensive Android application that helps people reconnect with lost items by allowing users to report lost or found items with **precise location data**, **interactive maps**, and **radius based search capabilities**.

## App Overview

The Lost and Found App connects lost items with their owners through an intuitive interface. Users can create adverts for lost or found items, upload photos, and leverage **GPS and mapping features** to pinpoint exact locations. The app features category filtering, timestamp tracking, and the ability to remove resolved adverts.

### Task 9.1P - New Geo Features

This version adds powerful location-based features:

- **Location Autocomplete** - Search and select addresses using Google Places API
- **Get Current Location** - Automatically use your GPS position
- **Interactive Map** - View all items as color-coded markers on Google Maps
- **Radius-Based Search** - Find items within X kilometers of your location
- **Location Picker** - Tap anywhere on map to select coordinates
- **Distance Calculation** - Shows how far each item is from current location

### ✅ Task 7.1P - Core Features

- **Create Adverts** - Post lost or found items with images
- **Image Upload** - Attach photos to each advert 
- **Category Filtering** - Filter by Electronics, Pets, Wallets, Documents, Other 
- **Timestamp Display** - Shows relative time  
- **Remove Adverts** - Delete items once resolved with confirmation 
- **SQLite Database** - Persistent local storage 

## 🛠️ Tech Stack

- **Language** - Java 
- **Minimum SDK** - API 21 (Android 5.0) 
- **Target SDK** - API 33 (Android 13) 
- **Database** - SQLite
- **Maps & Location** - Google Play Services (Maps & Location) 
- **Places API** - Google Places SDK for Android 
- **Image Loading** - Glide 
- **UI Components** - Material Design Components
